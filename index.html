~ Description ~


So, this game is for 2 players, each click changes the player's turn. The board has 10x10 cells, so each player can make 50 moves during the one game. The main strategy is to get 5 symbols in a row, or in a column, or in the diagonal lines. The game also ends with draw in case, if all of the cells have symbols. Let's play tic tac toe!
